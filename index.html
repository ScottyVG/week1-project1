What is JavaScript?

JavaScript (JS) is a programming language used to make web pages interactive.
JavaScript was first introduced in 1995 as a way to add programs to the Netscape Navigator web browser. It has since been adopted by all other major browsers, and is the basis for modern web applications.
The interactivity runs smoother on a web page as page reloads are not needed for every action since the JavaScript is downloaded to the user's computer and then processed within the browser.
Apps like Gmail, Google Maps, and Facebook are possible because of JS.
When JavaScript was first introduced, Java was being heavily marketed and adopted, and JavaScript sought to ride on the success. Now we are stuck with the name.



The goal for this course is to establish a solid foundation of:

  • how the language is structured
  • learn how to use JavaScript to create an interactive DOM
  • integrate external sites and data
  • successfully deploy an application (or two).

Outputting JavaScript

JavaScript can "display" content in different ways:

  • Writing into an alert box, using window.alert().
  • Writing into the HTML output using document.write().
  • Writing into an HTML element, using innerHTML.
  • Writing into the browser console, using console.log().



window.alert()
An alert box pops up over the browser and can display information. The user will have to click "OK" to close the alert box and this is most often used to make sure the user sees the message.

alert("I am an alert box!");

document.write()
This method is useful for testing purposes and adding JS directly into an HTML page.

document.write("I'm text added directly to the web page using JS.");

Be careful when using document.write() as when it is called after an HTML document is fully loaded and will delete all existing HTML.

innerHTML
JavaScript can be added to the web page by targeting an ID.

document.getElementById("demo").innerHTML = 'JS content can be added to a unique ID';



Make sure you check the console for your code.
